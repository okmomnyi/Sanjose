How to Use This Tool

  1. Open this page in a modern web browser (Chrome, Firefox, Edge, Safari)
  2. Click the "Convert Header Logo to PNG" button to convert the main logo
  3. Click the "Convert Favicon to PNG" button to convert the favicon
  4. Each click will automatically download the PNG file
  5. The downloaded files will be saved to your browser's default download location
  6. Move the downloaded PNG files to the appropriate folders in your project

Note: For best results, right-click on the preview images and select "Save image as..." to manually save them as PNG files.
